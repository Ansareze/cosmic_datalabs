# Assets Folder

This folder is for storing images and other media assets for the Cosmic DataLabs website.

## Suggested Images to Add:

1. **Logo** - `logo.png` or `logo.svg` (replace emoji logo)
2. **Certificate Mockup** - `certificate-preview.png` (for certificate section)
3. **Social Media Icons** - Individual icon files for WhatsApp, LinkedIn, Instagram
4. **Favicon** - `favicon.ico` or `favicon.png`

## Current Status:

Currently, the website uses emoji icons and CSS graphics. You can replace these with actual image files by updating the HTML and CSS accordingly.
